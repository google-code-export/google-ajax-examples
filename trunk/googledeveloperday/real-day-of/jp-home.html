会場およびスケジュール

Google Codeをはじめよう

  • Google AJAX Search API - Google 検索の結果にfrom JavaScript アプリケーションからアクセス
  • Google AJAX Feed API - ダウンロードして、JavaScriptでpublic feedsをマッシュアップ
  • Google Data APIs - Google カレンダーやBloggerといったGoogleのサービスとデータの読み込み、書き込み
  • Google Gadgets - iGoogleやGoogle デスクトップで使えるシンプルなミニ アプリケーションの作成
  • New! Google Gears - Webブラウザ上で、オフラインのAJAXアプリケーションが可能に
  • New! Google Mapplets - Google マップにミニアプリやレイヤーを追加しよう
  • Google Maps API - Google マップをあなたのウェブページに組み込み
  • Google Web Toolkit - Java言語でAJAX アプリケーションの作成
