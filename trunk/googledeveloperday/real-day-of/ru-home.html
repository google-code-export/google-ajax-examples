Начните использовать Google Code

  • Новинка! Google Gears - используйте AJAX-приложения офлайн с помощью веб-браузера.
  • Новинка! Google Mapplets - создавайте мини-приложения и слои для Карт Google.
  • Google AJAX Search API - используйте результаты веб-поиска Google для приложений на JavaScript.
  • Google AJAX Feed API - загружайте и используйте открытые потоки в собственных программах на JavaScript.
  • Google Data APIs - читайте и записывайте информацию в сервисах Google, в том числе в Календаре и Blogger.
  • Гаджеты Google - создавайте простые мини-приложения для главной страницы Google и Google Desktop.
  • Google Maps API - встраивайте Карты Google в собственные веб-страницы.
  • Google Web Toolkit - пишите AJAX-приложения на Java.
