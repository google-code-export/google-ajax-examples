Einführung in Google Code

  • Google AJAX Search API - Mit Javascript-Applikationen auf Google Suchergebnisse zugreifen.
  • Google AJAX Feed API - Mashups mit Javascript aus öffentlichen Feeds erstellen.
  • Google Data APIs - Daten für Google Services wie z. B. Google Kalender und Blogger lesen und schreiben.
  • Google Gadgets - Mini-Applikationen für die Google Homepage und Google Desktop schreiben.
  • Neu! Google Gears - ermöglicht offline AJAX-Anwendungen im Webbrowser.
  • Google Maps API - Google Maps in eigene Websites einbinden.
  • Neu! Google Mapplets - Erstellen Sie Mini-Anwendungen und individuelle Ebenen für Google Maps.
  • Google Web Toolkit - AJAX-Applikationen mit Java.
