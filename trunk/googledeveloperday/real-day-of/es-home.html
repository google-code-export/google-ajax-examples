Empieza a utilizar Google Code

  • Google AJAX Search API - Accede a los resultados de búsqueda de Google desde aplicaciones JavaScript.
  • Google AJAX Feed API - Descarga y crea mashups de feeds públicos en JavaScript.
  • Google Data APIs - Lee e incluye datos en los servicios de Google, incluidos Google Calendar y Blogger.
  • Google Gadgets - Crea miniaplicaciones sencillas para la página de inicio de Google y Google Desktop.
  • ¡Nuevo! Google Gears - Utiliza aplicaciones AJAX offline en un navegador web.
  • Google Maps API - Inserta los mapas de Google en tus propias páginas web.
  • ¡Nuevo! Google Mapplets - Crea miniaplicaciones y capas para Google Maps.
  • Google Web Toolkit - Crea aplicaciones AJAX en Java.
