体验 Google Code

  • Google AJAX Search API - 在 JavaScript 应用中获取 Google 搜索结果。
  • Google AJAX Feed API - 在 JavaScript 中下载或展示 Atom 和 RSS Feeds。
  • Google Data APIs - 从 Google Calendar，Blogger 等服务中读写数据。
  • Google Gadgets - Google Gadgets – 为 Google 个性化首页和 Google 桌面编写小工具。
  • 最新！ Google Gears - 允许 AJAX 在网页浏览器离线使用。
  • Google Maps API - 将 Google 地图嵌入到自己的网页中。
  • 最新 ! Google Mapplets - 在 Google 地图上创建小应用和图层。
  • Google Web Toolkit - 使用 Java 语言编写 AJAX 应用。
